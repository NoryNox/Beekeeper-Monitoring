# Beekeeper Monitoring

Die ultimative KI-App für Imker – erkennt 6 Bedrohungen in Echtzeit:

- Hornisse
- Varroa-Milbe
- Drohnenbrut
- Pollenmangel
- Schwarm
- Königin

**100% offline · 100% anonym · 100% kostenlos**

## So benutzt du die App

1. Öffne die App: https://norynox.github.io/Beekeeper-Monitoring/
2. Gib deine Handynummer ein → Speichern
3. Kamera starten → auf den Stock richten
4. Bei Bedrohung: Rote Box + SMS an deine Nummer

## Installation als App

- Chrome/Safari → 3 Punkte → „Zum Startbildschirm hinzufügen“
- Läuft komplett ohne Internet

## Funktionen

- Registrierung (lokal gespeichert)
- Dark Mode
- Logs exportieren
- Echte KI-Erkennung (YOLOv8n)
- SMS-Alarm per E-Mail-to-SMS

**Keine Daten werden gespeichert oder übertragen – alles bleibt auf deinem Gerät!**
